# dotfiles
dot configuration files of various kinds (vim, fish shell, tmux...)

## Install
Use `ln -s` to link the files in this directory to the places where the files should be to be loaded by the respective programs:

Here is my setup

- fish: `~/.config/fish/`
- i3: `~/.config/i3/`
- nvim: `~/.config/nvim/`
- vim: `~/.vim`
- vimrc: `~/.vimrc`
- tmux.conf: `~/.tmux.conf`
